Web interface for gopro hero2

Instructions

  1. power on gopro and wifi bacpac
  2. set wifi bacpac in "phone and tablet" mode
  3. connect your computer to your gopro
  4. verify your ip is 10.5.5.109 ( if not force it )
  5. fill the input with bacpac wifi password
  6. run command below
